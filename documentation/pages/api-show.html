• show(i)
  • Returns:

    this

Parameter Type Optional Description
i integer Yes Index of the node to show, if not passed the function will be executed for all nodes.

This function let you to show elements.

HTML 
    
DIV1

    
DIV2

    
DIV3
Javascript 
    // create a jPepper object for all DIVS
    var divs = _("DIV");     

    // let's show all divs
    divs.show();

    // let's show only the second one
    divs.show(1);

The style property display will be setted to "".
