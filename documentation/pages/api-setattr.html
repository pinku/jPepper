• setAttr(attr, val, i)
  • Returns:

    this

Parameter Type Optional Description
attr string No Attribute's name to set.
val string No Attribute's value to set
i integer Yes Index of the node, if not passed the function will be executed for all nodes.

This function let you to set attribute value for elements.

DIV1
Javascript 
    // set a data- attribute
    _("DIV").setAttr("data-id","xyz");
