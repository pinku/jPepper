• removeAttr(attr, i)
  • Returns:

    this

Parameter Type Optional Description
attr string No Attribute's name to remove.
i integer Yes Index of the node, if not passed the function will be executed for all nodes.

This function let you to remove an attribute from elements.

DIV1
Javascript 
    // remove a data- attribute
    _("DIV").removeAttr("data-id");
