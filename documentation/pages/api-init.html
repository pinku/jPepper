• jPepper.init(sel)
  • Returns:

    this

Parameter Type Description
sel string A string containing selectors expression

jPepper.init(sel) function is a static function, but it's not intended for direct calling. It's always called from jPepper(sel) main function.

It's the real jPepper object and at each jPepper(sel) call a new istance of this will be created.
