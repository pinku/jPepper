• off(events)
  • Returns:

    this

Parameter Type Optional Description
events string No Name of the event/s to bind. It can be a single event name (es: click) or multiple events, in this case separate each event with a space (es: click tap)

This function let you to unbind events from elements.

You can unbind a single event like in the example below:

HTML 
    Click me!
Javascript 
    _("#mybutton").on("click", function(e){
        alert("Ok ok! you clicked me but this is the last time!");
        _(e.currentTarget).off("click");
    });

Or you can unbind all events at the same time:

Javascript 
    _("a").off();
