• getStyle(style, i)
  • Returns:

    value of the css inline style (if found)

Parameter Type Optional Description
style string No Name of the css inline style to search.
i integer Yes Index of the node, if not passed the function will be return the first node's style value.

This function let you get css inline style value for a element.

HTML 
    
DIV1
Javascript 
    var color = _("DIV").getStyle("color");     
    alert(color);    // red
