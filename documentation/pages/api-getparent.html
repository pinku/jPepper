• getParent(i)
  • Returns:

    new jPepper.init() object with parents of the elements.

Parameter Type Optional Description
i integer Yes Index of the node, if not passed the function will be return the first node's attribute value.

This function let you create a new jPepper.init() object with the parents of the elements.

HTML 
    
I'm daddy!
    

            I'm a child
Javascript 
    var child=_("#child");
    var daddy = child.getParent();  // return a new jPepper.init() object with 
                                    // only div#daddy
Javascript 
    var divs=_("div");
    var parents = divs.getParent(); // return a new jPepper.init() object with 
                                    // all the parents  of the elements
