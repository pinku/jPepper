• on(events, fn, args)
  • Returns:

    this

Parameter Type Optional Description
events string No Name of the event/s to bind. It can be a single event name (es: click) or multiple events, in this case separate each event with a space (es: click tap)
fn function No Handler to connect to the event.
args object Yes Object that will be passed as argument to the handler function.

This function let you connect a handler function to events.

You can connect a single event like in the example below:

HTML 
    Click me!
Javascript 
    _("#mybutton").on("click", function(e){
        alert("Hei who clicked me?!?");
    });

Or you can connect multiple events at the same time:

Javascript 
    _("#mybutton").on("click tap", function(e){
        alert("Hei who clicked me?!?");
    });

You can connect more of the same event to an element, in this case all handlers will be called in binding sequence.

Javascript 
    // first click handler
    _("#mybutton").on("click", function(e){
        alert("I'm the first handler!");
    });
    // second click handler
    _("#mybutton").on("click", function(e){
        alert("I'm the second handler!");
    });

You also can pass some arguments to the event handler.

Javascript 
    for (var i=0; i < 10; i++) {
        _("a#nr" + i).on("click", function(e, a) {
            alert("My index is: " a.index);
        }, 
        {
            index: i
        });
    }
