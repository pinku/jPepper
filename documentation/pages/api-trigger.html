• trigger(events)
  • Returns:

    this

Parameter Type Optional Description
events string Yes Name of the event/s to bind. It can be a single event name (es: click) or multiple events, in this case separate each event with a space (es: click tap) You can also use namespace to make unique you binging, add a dot and the your custom name for the event(ex: click.myclick)

This function let you cann handler connected to events.

HTML 
    Click me!

You can trigger a single event by event name:

Javascript 
    var a = _("#mybutton");
    a.on("click", function(e){
        alert("Ok ok!");
    });
    a.trigger("click");

You can trigger all events:

Javascript 
            var a = _("#mybutton");
    a.on("click", function(e){
        alert("Ok ok!");
    });
    a.trigger();

You can trigger a event by custom name:

Javascript 
    var a = _("#mybutton");
    a.on("click.first", function(e){
        alert("first!");
    });
    a.on("click.second", function(e){
        alert("second!");
    });
    a.trigger(".second"); // or a.trigger("click.second");
