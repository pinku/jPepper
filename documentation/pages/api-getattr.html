• getAttr(attr, i)
  • Returns:

    value of the attribute (if found)

Parameter Type Optional Description
attr string No Attribute's name to set.
i integer Yes Index of the node, if not passed the function will be return the first node's attribute value.

This function let you get attribute value for a element.

HTML before 
    
DIV1
Javascript 
    var attr = _("DIV").getAttr("data-id");     
    alert(attr);    // myid
