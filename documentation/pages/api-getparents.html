• getParents(sel, i)
  • Returns:

    new jPepper.init() object with parents at any level of the elements.

Parameter Type Optional Description
sel string No A string containing selectors expression
i integer Yes Index of the node, if not passed the function will be return the first node's attribute value.

This function let you create a new jPepper.init() object with the parents at any levels of the elements.

HTML
Javascript 
    var spans = _("span");
    var divs = spans.getParents("div"); // will return a new jPepper.init() object 
                                        // containing div1 and div2
