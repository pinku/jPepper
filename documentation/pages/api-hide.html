• hide(i)
  • Returns:

    this

Parameter Type Optional Description
i integer Yes Index of the node, if not passed the function will be executed for all nodes.

This function let you to hide elements.

HTML 
    
DIV1

    
DIV2

    
DIV3
Javascript 
    // create a jPepper object for all DIVS
    var divs = _("DIV");     

    // let's hide all divs
    divs.hide();

    // let's hide only the second one
    divs.hide(1);

The style property display will be setted to "none".
