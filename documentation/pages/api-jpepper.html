• jPepper(sel) = _(sel)
  • Returns:

    new jPepper.init(sel)

Parameter Type Optional Description
sel string No A string containing selectors expression

jPepper(sel) function searchs the DOM for any elements that match selectors.

It can be written also in the short way using the _(underscore) character.

    // long way
    var el1 = jPepper("DIV");
    
    // short way
    var el2 = _("DIV");

Selectors can be simple, like ID, css class and attributes

    // ID
    var el1 = _("#myid");

    // css class
    var el2 = _(".myclass");

    // attribute
    var el3 = _("[data-myattr=123]");

Or can be complex and nested

    var el1 = _("body section.main a[data-attr=123]");
