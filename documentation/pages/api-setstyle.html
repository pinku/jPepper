• setStyle(style, val, i)
  • Returns:

    this

Parameter Type Optional Description
style string No Name of the css style property to set.
val string No Value to set for the style property.
i integer Yes Index of the node, if not passed the function will be executed for all nodes.

This function let you to set an inline css style property for elements.

DIV1
Javascript 
    // set color style
    _("DIV").setStyle("color","red");
